Responsive CSS Slider Version 2

This is a completely CSS-based, responsive slider. Its a cleaned-up, leaner, more svelte version of my previous CSS slider. In this version I've used SCSS to create an external library for most of the slider controls/logic and I've left only styling to be done in the main css file.

One

The nice thing about this slider is that it's completely content-agnostic, though if you want to have a background it looks best if you give the slides a fixed height!
