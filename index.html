Responsive CSS Slider Version 2

This is a completely CSS-based, responsive slider. This version uses all of the same principles that my previous version did, however I've switched to the translate3d property to allow for proper GPU rendering - thus helping devices with lower processor speeds to render the transition smoothly.

One

The nice thing about this slider is that it's completely content-agnostic, though if you want to have a background it looks best if you give the slides a fixed height!
